Shell del dashboard

La estructura base vive en dashboard-layout.component.html

El layout principal del template esta resuelto con un shell basado en grid: header arriba, sidebar a la izquierda y una columna derecha donde conviven main y footer. Esta pagina sirve para explicar esos puntos de extension sin mezclarlo con componentes reutilizables.

Archivo clave

src/app/layout/dashboard-layout/dashboard-layout.component.html

Mapa rapido

Que hace cada zona del layout

Antes de tocar el shell conviene ubicar donde vive cada pieza. Asi puedes modificar el layout sin entrar a ciegas al proyecto.

Header

Branding, acciones globales y boton hamburguesa para mobile.

Sidebar

Navegacion lateral del template. En desktop ocupa su columna y en mobile aparece como overlay.

Main

Area principal donde se renderizan las paginas privadas y vive el scroll del contenido.

Footer

Pie de la columna derecha. Es opcional y puede retirarse sin romper el shell.
