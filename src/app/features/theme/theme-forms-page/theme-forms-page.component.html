Composicion de formularios

En este template los formularios se ordenan con grid, flex y breakpoints de Tailwind

En lugar de recrear componentes tipo Bootstrap para filas y columnas, la idea es usar utilidades del layout junto con los shared del proyecto. Asi mantienes flexibilidad y el usuario aprende patrones reales de composicion para dashboards.

Regla base

grid-cols-1 + breakpoints + gap consistente

Criterio del template

La guia Theme / Forms existe para ensenar composicion real con Tailwind y con los componentes shared. No se busca recrear una API tipo Bootstrap para filas y columnas, sino dejar patrones claros que puedas copiar y adaptar en proyectos nuevos.
