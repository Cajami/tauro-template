Guardar pantalla completa
Al hacer clic, veras el bloqueo real de toda la pantalla central del dashboard. Esta demo ya no usa un bloque local: dispara el loading global del `main` para que puedas validar el comportamiento real del layout.
Recomendacion de uso: el `loading` del boton sigue siendo correcto para acciones cortas como `Guardar`, pero no reemplaza un overlay. Si el usuario ya no debe tocar una seccion o toda la vista central mientras esperas el API, este componente es la pieza adecuada.

Integracion recomendada para el `main`

Para bloquear toda la pantalla central del dashboard, el template renderiza un overlay absoluto dentro del `main` del `DashboardLayout` y controla su estado desde `MainLoadingService`. Asi el mensaje siempre queda centrado en la parte visible del `main`, aunque la pagina tenga mucho scroll, y el sidebar o header no se tocan.
